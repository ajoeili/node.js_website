Variables, Datatypes and Functions

Variables store data in programming languages. Data types define the kind of data that can be stored in a variable, such as numbers, strings, booleans, and objects. Functions are reusable blocks of code that perform specific tasks.

Variables and Data Types

  • Variables store values that can be used later in a program.
  • Numbers represent numeric values.
  • Strings represent text.
  • Booleans represent true or false.
  • Objects store collections of related data.

Functions

  • Functions are reusable blocks of code.
  • They help organize and structure programs.
  • Functions can accept parameters and return values.

Callback Functions

Callback functions are functions that are passed as arguments to other functions and executed after a certain event or condition occurs. They are commonly used in asynchronous programming.

Advanced JavaScript Functions

  • Higher-order functions can take other functions as arguments.
  • They can also return functions as their result.
  • This enables more functional programming patterns in JavaScript.

Array Loop Methods

  • map() – transforms each element in an array.
  • filter() – selects elements based on a condition.
  • reduce() – combines values into a single result.

These methods are often used in a functional programming style when working with arrays.

Type Coercion

Type coercion is JavaScript’s automatic conversion of values from one type to another, and can lead to unexpected results if not handled carefully. It is therefor good to use strict comparisons (===) which can help avoid many issues.

Fetch Requests

The fetch() function is used to make network requests to retrieve data from a server, and is commonly used when working with APIs.

Date and Time

JavaScript provides built-in objects like Date that allow you to create, manipulate, and format dates and times.

Scoping (const vs let)

  • let allows variables to be reassigned.
  • const creates variables that cannot be reassigned.
  • Both use block scope, meaning they only exist inside the block they are declared in.
